# Plan: Claw Agency Landing Page REBUILD

## Overview
Reconstruct the Claw Agency landing page from scratch with a "Functional Clarity" approach (n8n/Apple style). Focused on direct visual communication for Australian Tradies.

## Project Type
WEB

## Success Criteria
- [ ] Language: 100% Australian English (G'day, No Drama).
- [ ] Design: Functional Clarity (Stark White / Charcoal / Electric Orange).
- [ ] Typography: Plus Jakarta Sans for all elements.
- [ ] Hero: Massive centered headline + iPhone Mockup (Tradie IG Profile).
- [ ] Workflow: n8n-style visual flow: `Customer DM → Claw AI → Job Booked`.
- [ ] 🔴 Purple Ban: Absolute zero purple/violet.

## Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion (Subtle & Functional)
- **Fonts:** Plus Jakarta Sans (Google Fonts)
